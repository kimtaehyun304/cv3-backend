### 실행 방법
1. download ZIP 이후 압축해재
2. VSCODE에서 압축해재한 폴더로 이동
3. VSCODE 터미널에서 아래 명령어 입력
4. npm install
5. npm run dev

프론트, 백엔드 실행 방식 동일합니다.  
직접 실행해도 동작하지만, 심사 편의를 위해 배포했습니다.
